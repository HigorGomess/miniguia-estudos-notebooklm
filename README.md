# miniguia-estudos-notebooklmm
# 📚 Miniguia de Estudos com NotebookLM

## 📌 Contexto

Este projeto foi desenvolvido como parte de um desafio da DIO com o objetivo de explorar o uso da Inteligência Artificial como ferramenta de aprendizagem ativa.

O tema escolhido foi **Inteligência Artificial e Engenharia de Prompts**.

---

# 🎯 Objetivos

- Entender o funcionamento das IAs generativas
- Aprender técnicas de engenharia de prompts
- Utilizar IA para organizar estudos
- Criar um material de revisão reutilizável

# 🧠 Testes de Prompts

## Prompt 1

O que são IAs generativas?

Resposta:

IAs generativas são modelos de inteligência artificial capazes de criar novos conteúdos como textos, imagens, códigos e músicas a partir de dados usados no treinamento.

---

## Prompt 2

Qual a diferença entre chatbots antigos e modernos?

Resposta:

Os chatbots antigos funcionavam com regras pré-programadas.  
Os chatbots modernos utilizam IA generativa e aprendizado de máquina para criar respostas dinâmicas.

---

# ⚠️ Dificuldades

Durante os testes foi possível perceber que:

- Prompts muito curtos geram respostas vagas
- Perguntas mais detalhadas produzem respostas melhores
- Dividir a pergunta em etapas melhora a qualidade da resposta

# 📘 Miniguia de Estudo

## O que é IA Generativa?

IA generativa é um tipo de inteligência artificial capaz de criar novos conteúdos com base em padrões aprendidos em grandes conjuntos de dados.

Ela pode gerar:

- textos
- imagens
- códigos
- músicas
- vídeos

---

# 📚 Glossário

Prompt → Instrução dada para a IA  
IA Generativa → IA capaz de criar conteúdos novos  
Chain of Thought → Técnica que faz a IA explicar passo a passo  

---

# 🧩 Prompts Reutilizáveis
